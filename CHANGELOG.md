# Changelog

## [1.0.0] - 2026-06-01

### Added
- **Core:** Laravel 9 API foundation.
- **Auth:** JWT stateless authentication via `php-open-source-saver/jwt-auth`.
- **Database:** Migrations and seeders for `users` and `students`.
- **Features:** Complete CRUD for students with Eloquent SoftDeletes.
- **Security:** Strict Form Request validation.
